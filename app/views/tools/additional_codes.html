What are additional codes?

Additional codes are used on the tariff for a number of purposes. The first character of the additional code determines the type of activity it is intended to influence.

First character Meaning
2 These codes influence duties, providing preferential tariff treatment under certain circumstances
3 These codes impact whether or not a prohibition is to be applied
4 These codes impact whether or not a restrictive measure is to be applied
7 Meursing codes (Northern Ireland tariff only)
8, A, B or C These codes relate to exporting company for determining applicable anti-dumping or anti-subsidy duty.
V These codes impact which VAT rate is due
X These codes impact which excise rate is due

To search for additional codes, and the measures to which they apply, enter either a four-digit additional code or words from the additional code in the fields below.
