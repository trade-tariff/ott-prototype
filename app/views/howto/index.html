Using the Integrated Tariff

The UK Integrated Tariff (UKIT) is the complete set of customs duties and controls that can apply to imports and exports from the UK.

Structured around a commodity code nomenclature it sets out the:

  • The standard duty rates, aka the UK Global Tariff (UKGT).
  • Preferential rates, which provide for country specific lower rates as agreed with the UKs trading partners
  • etc.
